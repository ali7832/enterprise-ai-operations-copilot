# Enterprise AI Operations Copilot Frontend

React operator console for the FastAPI backend.

## Local Run

```bash
npm install
npm run dev
```

The dev server runs on `http://127.0.0.1:4173` and proxies the backend routes to `http://127.0.0.1:8000`.

If you want to point it at another backend, set a value in the API Base field inside the UI.
